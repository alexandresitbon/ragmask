Sizing

This page is under development.

When printing the first page of the instructions, set the scale in your print settings to the following:

  • Kids - 90%
  • Small - 95%
  • Medium - 100%
  • Large - 105% (will need larger material cut)

Note that you should adjust the width of the first stitch by a proportional amount (and can trim the extra). If making a lot of the same pattern, adjust the size of the original material cuts to minimize waste.
